Flipkart Automation:
================

* It is just automation demo of Flipkart website

* place the project in your local system

The Test Automation Framework allows you to create the tests for Web Applications which can be run on different browsers and Operating systems but 
this tests are limited to chrome browser and  windows Operating system.


Pre-requisite Tools and Technologies
============================
Java JDK 1.8 Above

Java IDE like Eclipse, Netbeans or IntelliJ IDEA

Maven Environment (Should be present in Netbeans and IntelliJ IDEA out of the box. For Eclipse, plugin might be required to be installed)


Technology Stack
==============
Java JDK 1.8 above
Maven 3.5 above
TestNG 7.0.0
Selenium WebDriver 3.14
Extent Report

Setup Info
=========
save the project in your local drive , Open the folder where pom.xml is located and run the following command and reports are genarated in TestReport folder

Run the command:
==============
mvn clean install
